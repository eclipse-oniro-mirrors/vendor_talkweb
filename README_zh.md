# vendor_talkweb

## 简介

该仓库托管拓维信息系统股份有限公司官方开发的产品样例代码，主要包括开发板案例代码、配置及文档。

## 目录结构

代码路径说明。

```
vendor/talkweb/
├── LICENSE
├── niobe407
│   ├── BUILD.gn
│   ├── config.json     # config文件
│   ├── hals            # 适配层目录
│   └── kernel_configs  # 内核配置
├── OAT.xml
└── README_zh.md
```
